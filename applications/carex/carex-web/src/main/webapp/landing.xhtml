Clinic Registration

Register the clinic once. CareX will create the organization, license, and clinic admin login.

This flow connects directly to the backend organization and license records, so your clinic is provisioned using the same core administration logic.

India gateway recommendation: Razorpay is the best fit to start with because it supports test-mode cards, UPI, and mock bank flows without real money movement. The payable amount now comes from the shared India pricing configured in Application Management.
